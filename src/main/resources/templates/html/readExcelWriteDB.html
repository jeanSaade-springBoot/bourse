How Insert Works
  • Use this option to add new data.
  • If data already exists for the same date, the upload will stop.
  • To modify existing records, please use Update Existing Data.
How Update Works
  • Select the columns you want to update.
  • Only selected columns will be modified.
  • Non-selected columns will remain unchanged.
  • Existing data for the same date will be updated.
  • If no existing data is found for the same date, please use Insert New Data.
  • Empty values in selected columns will be saved as blank.
